# Deric Ndikumana - Full Stack Web Developer Portfolio

A modern, responsive, dark-mode enabled portfolio website showcasing my skills, projects, and internship journey as a Full Stack Web Development intern at Future Interns.

## 🌐 Live Preview

Visit the portfolio: [https://dericndikumana.github.io/FUTURE_FS_01](https://dericndikumana.github.io/FUTURE_FS_01)

## ✨ Features

- **Dark/Light Mode Toggle** - Seamless theme switching with localStorage persistence
- **Responsive Design** - Fully optimized for mobile, tablet, and desktop screens
- **Custom Animations**
  - Animated cursor with follower effect
  - Scroll reveal animations
  - Skill bar animations
  - Floating orbs and gradient effects
  - Fade-up hero animations
- **Interactive Contact Form** - Powered by FormSubmit with success/error handling
- **Skill Showcase** - Visual skill bars with proficiency percentages
- **Project Gallery** - Showcasing real-world projects built during internship
- **Internship Tracker** - Progress indicator for internship tasks
- **Social Links** - Email, Phone, WhatsApp, and GitHub integration

## 🛠️ Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom styling with animations and gradients
- **JavaScript (Vanilla)** - Interactive features
- **Tailwind CSS** - Utility-first CSS framework

### Tools & Libraries
- **Font Awesome** - Icon library (v6.5.0)

## 🚀 Getting Started

### View Locally
Simply open `index.html` in any modern web browser:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

Edit the following in `index.html`:
